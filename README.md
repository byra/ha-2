# Hilton Assessment 2

## How to use

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn run dev
```

Install it and test:

Reports including code coverage and test results can be viewed from HTML or Terminal
```bash
npm install
npm run test
# or
yarn
yarn run test
```

Install it and build & start:

```bash
npm install
npm run build
npm run start
# or
yarn
yarn run build
yarn run start
```
